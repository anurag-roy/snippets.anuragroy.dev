---
title: 'Add Inter as a variable font'
tags: ['CSS', 'Font', 'Inter', 'Variable Font']
---

Paste the following code in the global CSS file:

```css
@font-face {
  font-family: 'Inter';
  font-weight: 100 900;
  font-display: swap;
  font-style: normal;
  font-named-instance: 'Regular';
  src: url('/fonts/Inter.var.woff2') format('woff2');
}
```
